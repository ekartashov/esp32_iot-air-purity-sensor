## Development Environment 

### Requirements

- `pyenv` (with `pyenv-virtualenv`)
- VSCodium IDE
- `bash`

1. `python3 ./scripts/vsix-downloader.py`
2. Optionally create VSCodium profile
3. Install extension from `.vsix` file in `.vscode_exts` directory
4. Pathch with `./scripts/patch.bash`
5. Install clangd extension and install clangd as prompted
